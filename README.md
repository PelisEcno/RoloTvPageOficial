# RoloTV

RoloTV es una aplicación Android desarrollada por un grupo de estudiantes interesados en el desarrollo de software y el entretenimiento digital.  
Permite ver películas, series y animes desde una interfaz moderna, rápida y optimizada.

## Características principales

- Reproductor ExoPlayer optimizado  
- Catálogo de películas, series y animes con API dinámica  
- Diseño adaptado a dispositivos móviles  
- Integración con Firebase Realtime Database  
- Conexión a Cloudinary para manejo de videos  
- Estructura modular y carga eficiente

---

## Tecnologías utilizadas

- Android (Java)  
- Firebase Realtime Database  
- Cloudinary  
- Node.js (backend y firmas)  
- Sketchware Pro
